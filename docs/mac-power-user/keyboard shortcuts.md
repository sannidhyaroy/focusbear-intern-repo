# Keyboard and Productivity Shortcuts

## 20 Essential macOS Shortcuts

### System

1. **`Cmd + Space`:** Spotlight search. Launch apps, calculate, convert units, search files, and define words without touching the mouse.
2. **`Cmd + Tab`:** App switcher. Hold `Cmd` and press `Tab` to cycle forward, `` ` `` to cycle backward. Press `Q` while holding `Cmd` to quit the highlighted app without switching to it.
3. **`Cmd + ~`:** Switch between windows of the same app.
4. **`Cmd + H`:** Hide current app. Different from minimizing, the window is instantly hidden and instantly restored.
5. **`Cmd + Option + H`:** Hide all other apps. Instant focus mode without a dedicated focus tool.
6. **`Cmd + Ctrl + Q`:** Lock screen immediately.
7. **`Cmd + Shift + 3`:** Screenshot entire screen.
8. **`Cmd + Shift + 4`:** Screenshot selection. Add `Space` to capture a specific window with a drop shadow.
9. **`Cmd + Shift + 5`:** Screenshot and screen recording toolbar with all options including timer and mic toggle.
10. **`Ctrl + Cmd + Space`:** Emoji and symbol picker anywhere.

### Text Editing

1. **`Cmd + Left/Right`:** Jump to start/end of line. Works in every  text field on macOS.
2. **`Option + Left/Right`:** Jump word by word.
3. **`Cmd + Shift + Left/Right`:** Select to start/end of line.
4. **`Option + Shift + Left/Right`:** Select word by word.
5. **`Fn + Delete`:** Forward delete (delete character after cursor).  macOS keyboards lack a dedicated forward delete key.
6. **`Option + Delete`:** Delete entire word before cursor.
7. **`Cmd + Delete`:** Delete entire line before cursor in most apps.

### Window Management

1. **`Cmd + M`:** Minimize current window to Dock.
2. **`Ctrl + Up`:** Mission Control. Shows all open windows across  all spaces. `Ctrl + Down` shows all windows of the current app.
3. **`Ctrl + Left/Right`:** Switch between Spaces (virtual desktops).

## Bonus: Terminal-Specific Shortcuts

- **`Ctrl + A`** / **`Ctrl + E`:** Jump to start/end of line in terminal
- **`Ctrl + R`:** Fuzzy search command history (enhanced with `fzf`)
- **`Ctrl + L`:** Clear terminal (faster than typing `clear`)
- **`Ctrl + C`:** Interrupt current process
- **`Ctrl + D`:** Send EOF / exit shell
- **`Ctrl + Z`:** Suspend process to background (`fg` to resume)
