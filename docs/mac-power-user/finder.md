# Finder & File System Navigation

## Finder Tricks and Shortcuts

### Navigation

1. **`Cmd + Shift + G`:** Go to any folder path directly, supports tab completion
2. **`Cmd + [`** / **`Cmd + ]`:** Navigate back and forward in folder history
3. **`Cmd + Up`:** Go to parent folder
4. **`Cmd + Down`:** Open selected file or folder
5. **`Space`:** Quick Look preview without opening the file

### View Modes

1. **`Cmd + 1/2/3/4`:** Switch between Icon, List, Column, and Gallery views
2. **`Cmd + Option + S`:** Toggle sidebar visibility
3. **`Cmd + /`:** Toggle status bar
4. **`Cmd + Shift + .`:** Toggle hidden files visibility

### File Operations

1. **`Cmd + Delete`:** Move selected file to Trash
2. **`Cmd + Shift + Delete`:** Empty Trash
3. **`Option + Drag`:** Copy file instead of move
4. **`Cmd + D`:** Duplicate file in place
5. **`Return`:** Rename selected file
6. **`Cmd + C`** then **`Cmd + Option + V`:** Move file (cut and paste equivalent)

### Search and Tags

1. **`Cmd + F`:** Open Finder search with filter options (kind, date, size, etc.)
2. **Tags:** Right-click any file to assign color tags, then filter by tag in sidebar or search
3. **`Cmd + Option + Space`:** Open Finder search window directly from anywhere

### Hidden Files and System Navigation

1. **`Cmd + Shift + G`** in any open/save dialog navigate to hidden paths like `/usr/local` or `~/.config` directly
2. **`defaults write com.apple.finder AppleShowAllFiles true`** enable hidden files permanently via Terminal (toggleable with `Cmd + Shift + .` without restart)

### Pro Tips

1. **Drag folder to Terminal** automatically pastes the full path
2. **Path Bar** (`View > Show Path Bar`) shows full path at bottom of Finder window, double-click any segment to navigate there
3. **Spring-loaded folders** hold a file over a folder and it springs open, allowing deep navigation while dragging
