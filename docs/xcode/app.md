# Create Simple Swift macOS App

## Project

The Xcode project is in [`Sample App`](../../assets/Sample%20App).

## What it does

A minimal macOS app created from the Xcode macOS App template using SwiftUI. The default `ContentView` displays "Hello, World!" centered on screen.

Modified `ContentView.swift` to display a personalized greeting and the current date, demonstrating basic SwiftUI state and text rendering.

## Steps taken

1. `Xcode` > `File` > `New` > `Project` > `macOS` > `App`
2. Set product name to "Sample App", interface to SwiftUI, language to Swift
3. Modified `ContentView.swift` to customize the greeting
4. Built and ran on macOS target to verify
